# Tech Stack

- Frontend: React with TypeScript and CSS Modules
- Backend: Next.js (App Router) with TypeScript Server Actions
- Database: PostgreSQL with Prisma ORM
- File Storage: AWS S3 or Wasabi
- Auth: NextAuth.js with Role-Based Access Control (RBAC)
- Notifications: Resend (Email) (Node-Cron with Twilio (SMS))
- Deployment: VPS (Ubuntu/Docker) or on-primise server
